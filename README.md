# Introduction

Simple helper function that enables fuzzy search via [`fzf`](https://github.com/junegunn/fzf) in Alfred 5.

# Dependency

You will need to have `fzf` installed.
Nothing else.

# Installation

Either copy `alfzf.py` to your repository, or:

```bash
pip install .
```
